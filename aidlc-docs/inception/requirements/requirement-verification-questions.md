## Question 1

B) 레퍼런스 대비 기능 동등화 — 가장 영향력 있는 기능 도입: 스킬 합성 레이어(OMC), 매니페스트 기반 설치 프로필(ECC), 모델 티어 라우팅(OMC), 증거 기반 검증(OMC)

D) 전체 범위 — 모든 갭을 종합적으로 여러 이터레이션에 걸쳐 해결 (전체 로드맵 계획, 최우선 항목부터 구현)
